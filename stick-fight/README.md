# Stick Fight Arena

A 2D stick figure fighting game where you battle against an AI opponent!

## How to Play

1. **Select Your Fighter** - Choose from 5 different colored stick figures:
   - Crimson (Red)
   - Azure (Blue)
   - Neon (Green)
   - Gold (Yellow)
   - Magenta (Purple)

2. **Fight!** - Battle against an AI-controlled opponent

3. **Win or Lose** - Each fighter has 5 hearts. Reduce your opponent's health to 0 to win!

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left/right |
| SPACE | Jump |
| M | Punch |

## Features

- 5 unique colored fighters to choose from
- Smooth stick figure animations
- AI opponent with dynamic behavior
- Health system with 5 hearts each
- Victory and Game Over screens
- Restart functionality

## Getting Started

Simply open `index.html` in a web browser to play!
