# Sky Platform

A 3D platformer game built with Three.js featuring floating islands in the sky.

## How to Play

### Controls
- **WASD / Arrow Keys** - Move
- **Space** - Jump
- **Shift** - Sprint

### Objective
Navigate the floating platforms, collect coins, and reach the highest point! Don't fall into the void!

## Running the Game

Since this game uses ES modules, you'll need to serve it through a local server:

### Option 1: Using Python
```bash
# Python 3
python -m http.server 8000

# Then open http://localhost:8000 in your browser
```

### Option 2: Using Node.js
```bash
# Install serve globally
npm install -g serve

# Run the server
serve .

# Then open the provided URL in your browser
```
