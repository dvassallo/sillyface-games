# Bunny Simulator

Live the life of a bunny!

## Controls
- **Move:** WASD or Arrow Keys
- **Sprint:** Shift (consumes stamina)

## Goal
- Eat plants (green circles) to keep your hunger down and health up.
- Avoid predators (red circles).
- Survive as long as possible!

## Stats
- **Health:** Reaches 0, you die. Regenerates if you are well-fed.
- **Hunger:** Increases over time. If it reaches 100, you start losing health.
- **Stamina:** Used for sprinting. Regenerates when walking or standing still.
